# :books: Operating System <sub>운영체제</sub>

## :memo: 개요

- 운영체제(Operating System)란 사용자의 하드웨어, 시스템 리소스를 제어하고 프로그램에 대한 일반적 서비스를 지원하는 시스템 소프트웨어를 의미
- 이렇나 운영체제의 구조와 동작 알고리즘에 대한 방법을 정리
