### DFS & BFS의 차이점에 대해서 설명할 수 있나요?

|          | DFS                                | BFS                              |
| -------- | ---------------------------------- | -------------------------------- |
| 탐색기준 | 분기를 선택해서 다음 분기까지 탐색 | 인접한 정점을 모두 선택해서 탐색 |
| 장점     | 탐색 깊이가 깊을때 유리            | 탐색 깊이가 낮을때 유리          |
| 최단경로 | 보장하지 못함                      | 보장함                           |
| 자료구조 | 스택                               | 큐, 힙                           |
